# Electron React Boilerplate

### Dependencies
* @babel/core 
* @babel/preset-env 
* @babel/preset-react 
* css-loader 
* style-loader 
* sass-loader 
* sass 
* webpack 
* webpack-cli 
* babel-loader

### Includes
* h1 Text
* Button with click-to-show-notification

### Instruction
* git clone https://github.com/my-store/electron-react-boilerplate.git
* npm install
